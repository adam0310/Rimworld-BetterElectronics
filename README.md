# Rimworld-BetterElectronics

This is a mod for rimworld.

# Mod Description

Electronics will never breakdown!
Prevents random short circuits!
Prevents solar flare!

Incompatibility: None;
Old Save Compatible: Yes;
Safely Removable: Yes;

Translations:
English,
Polish
